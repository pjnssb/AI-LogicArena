# AI-LogicArena (AI 社交推理游戏)

这是一个由 AI 驱动的社交推理游戏平台，支持多种模式，并可通过 OpenAI 兼容接口连接不同的大语言模型进行对抗测试。

## 支持模式

- **谁是卧底（Undercover）**：玩家通过描述词语寻找卧底。
- **狼人杀（Werewolf）**：经典的社交推理游戏，支持多种神职角色。

---

## 1. 项目结构

- `main.py`：后端服务实现（FastAPI/Flask）。
- `static/`：静态前端文件。
  - `index.html`：游戏主界面。
  - `undercover.js`：谁是卧底引擎逻辑。
  - `werewolf.js`：狼人杀引擎逻辑。
- `requirements.txt`：项目依赖。

---

## 2. 环境准备

- Python 3.8+
- Node.js 18+（可选，用于前端开发）
- 现代浏览器（Chrome / Edge / Safari）

---

## 3. 快速启动

### A. 启动后端服务

在项目根目录下运行：

```powershell
pip install -r requirements.txt
python main.py
```

### B. 访问前端页面

后端启动后，通常可以通过以下地址访问：
`http://localhost:8000` (具体取决于 `main.py` 的配置)

---

## 4. 后端接口说明

前端会与以下接口进行交互：

- `POST /api/models`：获取可用模型列表。
- `POST /api/chat`：发送对话请求。

---

## 5. 狼人杀规则实现要点

- **夜晚顺序**：守卫 -> 狼人 -> 女巫 -> 预言家。
- **守卫**：不可连续两晚守同一人。
- **狼人**：支持两轮协商后统一落刀。
- **女巫**：支持解药和毒药结算，支持奶穿（同守同救）。
- **技能连锁**：支持猎人/白狼王/白痴等特殊角色技能结算。
